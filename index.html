Message Macon

Connect with your city,
one message at a time.

What is this?

Message Macon allows you to simply text a public works issue directly to Macon's SeeClickFix system. You can now help you improve your neighborhood at anytime from anywhere!

How does it work?

The app was built by Code for America and is completely open-source for anyone, or any city, to use. Feel free to check out the project on Github.

Send your issue to SeeClickFix at anytime.

Number

Be sure to save Message Macon's number to your phone, that way you can text an issue at anytime: 478‑246‑7248

Message

In order for your issue to saved to the SeeClickFix system, make sure to format your text in the following way: Address. Issue.

Confirmation

Once your message has been sent (and is in the correct format), you should receive a confirmation message thanking you!
